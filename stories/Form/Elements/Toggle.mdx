The `Toggle` component inherits (nearly) everything that the `Checkbox` component has. It is a `Form` component that can be used to give the end-user a choice. We have a bunch of props that are shown in the table below like `name`, `label`, `helperProps`, `formSelectorWrapperProps`, `helperText`, `labelPosition`, `hideLabel`, `labelOverflow`, `version`, `spacing`, `checked`, `disabled` and `className`

# Examples

Example code with state:

```jsx
const ExampleComponent = () => {
  const [checked, setChecked] = useState(false);

  return (
    <Toggle name="test" onChange={() => setChecked(!checked)} checked={checked} helperText="Example helper text">
      Example toggle
    </Toggle>;
  )
};
```
