The `Toggle` component inherits (nearly) everything that the `Checkbox` component has. It is a `Form` component that can be used to give the end-user a choice.

# Examples

Below you can find an exampe of the `Toggle` component. It can be controlled through the [Props table](#props) at the bottom.

<DocsStory id="stories-form-elements-toggle--toggle" />

Example code with state:

```jsx
const ExampleComponent = () => {
  const [checked, setChecked] = useState(false);

  return (
    <Toggle onChange={setChecked(!checked)} checked={checked} helperText="Example helper text">
      Example toggle
    </Toggle>;
  )
};
```
