The `FormSection` component has been created to align specific section of the page with some header and sub header.

# Examples

A code example of using form section component is as follows:

```jsx
<FormSection header="Form Section Header Text" subtext="Form Section Subheader">
  <InputWrapper
    helperText="Helper text for this field. Description should be short and not repeat the label"
    label="Label for this inputfield"
    name="input1"
    onChange={() => {}}
    type="text"
    value=""
  />
</FormSection>
```
