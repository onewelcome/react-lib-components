# useSpacing

Spacing helps create consistent spacing between elements in your application.
Margins and paddings can be set via the `spacing` component property.

4px (0.25rem) scaling factor is used in <Typography variant="body" tag="span" children="react-lib-components" />.

Spacing multiplies provided value via factor.

## Setting spacings to components

Components consume object as a `spacing` property:

```jsx
<Typography variant="body" spacing={{ margin: 4 }}>
  1rem margin
</Typography>
```

The above example sets the margin as `1rem` (`4 * 0.25rem`).

`spacing` configures beneath properties:

- padding
- paddingTop
- paddingBottom
- paddingLeft
- paddingRight
- margin
- marginTop
- marginBottom
- marginLeft
- marginRight

`margin` and `padding` respect single, double, triple and quadruple values provided as a string property or number value.
All others consume a single number or string value. See beneath example:

```jsx
<Typography variant="body" spacing={{ margin: '4 0 12 2' }}>1rem margin-top, 0 margin-right, 3rem margin-bottom, 0.5rem margin-left</Typography>
<Typography variant="body" spacing={{ margin: '4 0 12' }}>1rem margin-top, 0 margin-right & margin-left, 3rem margin-bottom</Typography>
<Typography variant="body" spacing={{ margin: '4 0' }}>1rem margin-top & margin-bottom, 0 margin-right & margin-left</Typography>
<Typography variant="body" spacing={{ margin: 4 }}>1rem margin</Typography>
<Typography variant="body" spacing={{ margin: '5' }}>1.25rem margin</Typography>
<Typography variant="body" spacing={{ paddingLeft: '6' }}>1.5rem padding-left</Typography>
<Typography variant="body" spacing={{ paddingRight: 7 }}>1.75rem padding-left</Typography>
```

### Supported components

For now, spacing is supported only in `<Typography/>` component.
