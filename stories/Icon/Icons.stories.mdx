# Icons

&#x200B;<Typography tag="span" children={projectName[0].toUpperCase() + projectName.substring(1)}/> provides [material](https://fonts.google.com/icons) like icons.

## Usage

Icons are build in <Typography tag="span" children={projectName}/>. To use icon you have to use <LinkTo kind="icon--page">`Icon`</LinkTo> component.

```jsx
import { Icon, Icons } from 'react-lib-components';
<Icon icon={Icons.Warning} />;
```
