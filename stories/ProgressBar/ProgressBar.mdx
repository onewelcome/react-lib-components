The `ProgressBar` component can be used whenever user has to wait for a particular action to finish.

It takes 100% width of its parent by default. Its styles can be adjusted using the `className` prop.

# Examples

```jsx
<div style={{ width: "500px" }}>
  <ProgressBar />
</div>
```
