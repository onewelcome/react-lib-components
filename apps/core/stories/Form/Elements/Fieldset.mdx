The `Fieldset` component is created to improve accessibility and give you an easier time with creating sections between parts of the form. The `title` prop will create a `legend` HTMLElement for you, which is hidden and used for screenreader purposes only. A `Typography` component that you can control through the `titleVariant` prop is rendered instead. You can also optionally hide the title by providing `true` to the `hideTitle` prop.

By default, the `Fieldset` component will propagate its `disabled` prop to any Form component nested inside of it. This includes the following components:

- Checkbox
- CheckboxWrapper
- FormControl
- FormHelperText
- FormSelectorWrapper
- Input
- InputWrapper
- Label
- Radio
- RadioWrapper
- Select
- SelectWrapper
- Textarea
- TextareaWrapper
- Toggle

If, for some reason, you want to handle all `disabled` props yourself, you can set the `disablePropagation` prop of the `Fieldset` component to `true`. This will disable this behavior.

# Examples

Below you will find an example `Fieldset` component:

<Canvas of={FieldsetStories.Fieldset} id="components-layout-fieldset--fieldset" />

And here's how that would look like in your code:

```jsx
const ExampleComponent = () => {
  const [prefix, setPrefix] = useState("mr");
  const [firstName, setFirstName] = useState("");
  const [date, setDate] = useState("");

  return (
    <Form style={{ padding: "20px", backgroundColor: "#F5F8F8" }}>
      <Fieldset legend="Name">
        <FormControl grid={3} align="top">
          <SelectWrapper
            name="prefix"
            label="Prefix"
            value={prefix}
            onChange={event => setPrefix(event.target.value)}
          >
            <Option disabled value={undefined}>
              Prefix
            </Option>
            <Option value="mr">Mr.</Option>
            <Option value="mrs">Mrs.</Option>
          </SelectWrapper>
          <InputWrapper
            onChange={event => setFirstName(event.target.value)}
            label="First name"
            errorMessage={"Error message"}
            error={false}
            value={firstName}
            helperText={"Helper text"}
            name="first-name"
            type="text"
          />
          <InputWrapper
            onChange={event => setDate(event.target.value)}
            label="Last name"
            errorMessage={"Error message"}
            error={false}
            value={date}
            helperText={"Helper text"}
            name="callback-time"
            type="datetime-local"
          />
        </FormControl>
      </Fieldset>
    </Form>
  );
};
```
